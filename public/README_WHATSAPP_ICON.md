# WhatsApp Icon

Place your `wp.png` WhatsApp icon image in this directory (`public/`).

The image should be:
- A WhatsApp logo/icon
- Recommended size: 20x20px or larger (will be scaled to 20x20px)
- PNG format with transparent background
- Green WhatsApp brand color

The file path should be: `public/wp.png`

You can download a WhatsApp icon from:
- https://www.flaticon.com/free-icon/whatsapp_733585
- Or use the official WhatsApp brand assets
- Or create your own icon

Once you add the file, it will automatically appear in the SHARE button on the demo page.
